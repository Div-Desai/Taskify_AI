# Taskify — Kanban-Driven Smart Productivity Workspace

A production-grade task management platform built with React, featuring role-based authentication, real-time state management, and a clean Kanban-style UI.

---

## Features

- 🔐 JWT authentication with role-based access control and session persistence
- 📋 Full CRUD for projects, tasks, and comments with optimistic UI updates
- 🔍 Search, filters, and pagination via URL params
- 📄 Validated forms using React Hook Form + Zod
- 🔄 Scalable state management with RTK + React Query + Context API
- 🛡️ Protected routes, nested routing, and dynamic params with React Router v6
- ⚡ Clean modular architecture built for production

---

## Tech Stack

| Category | Tech |
|---|---|
| Frontend | React 18, Vite |
| State | Redux Toolkit, React Query, Context API |
| Routing | React Router v6 |
| Forms | React Hook Form, Zod |
| Auth | JWT, Firebase |
| Styling | Tailwind CSS |
